## ✨ Overview

The **Network Monitoring Dashboard** is a practical IT operations tool built to simulate and visualize network performance, active device statuses, and system infrastructure health. It bridges core computer networking concepts with a clean, responsive web interface for efficient system tracking.

---

## 🚀 Key Features

- **📊 Visual Dashboard Interface:** Clean, structured UI for real-time monitoring and quick status evaluation.
- **💻 System & Device Tracking:** Monitor network device availability and operational health.
- **⚡ Real Network Checks:** Interactive scripts executing live network status checks and updates.
- **📈 Performance Metrics:** Track and visualize latency, connectivity states, and infrastructure data.
- **📱 Responsive Layout:** Fully optimized for seamless viewing across screens and administrative viewports.

---

## 🛠️ Built With

- **Core:** HTML5, Modern CSS3 (Grid, Flexbox)
- **Scripting:** Vanilla JavaScript (DOM manipulation, async network checks)
- **Concepts:** IT Infrastructure, Computer Networking, Systems Monitoring
